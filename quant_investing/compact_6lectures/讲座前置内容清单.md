# FEC 量化讲座前置内容清单

## 一、周四只讲 C01 时的最低前置要求

这一部分是新生在第一次来听课前，最好已经完成的内容。做不到全部也没关系，但至少要能打开 notebook 并运行代码。

### 1. 电脑与系统
- 自带笔记本电脑
- Windows / macOS 都可以
- 能连接校园网
- 能正常访问 Jupyter Notebook

### 2. 必装软件
- Python 3.10 或 3.11
- Jupyter Notebook
- VS Code 或 PyCharm（二选一即可，不强制）

### 3. 必装 Python 包
- `pandas`
- `numpy`
- `matplotlib`
- `jupyter`
- `rqdatac`

建议直接运行：

```bash
pip install pandas numpy matplotlib jupyter rqdatac
```

### 4. 必须会的最小操作
- 能打开命令行或终端
- 知道 `pip install ...` 是装包命令
- 能启动 Jupyter
- 能打开 `.ipynb` 文件
- 能运行单个 cell
- 知道报错时先看最后一行错误信息

### 5. 必须理解的最小概念
- 什么是 Python
- 什么是包（package）
- 什么是 notebook / cell
- 什么是变量
- 什么是 DataFrame
- 什么是“运行一个 cell”

### 6. 米筐 / RQData 账号要求
- 有可用的 `rqdatac` 教育版或其他可用账号
- 知道如何在 notebook 中填写 `PASSWD`
- 如果教育版要求教育网访问，要确保自己在校园网环境下

### 7. C01 课前建议预习
- 什么是股票、ETF、指数
- 什么是收盘价 `close`
- 什么是成交量 `volume`
- 什么是收益率
- 什么是波动率

## 二、按完整 6 节课准备的前置内容

如果你希望学生后面 6 节课都能跟上，建议他们在第一次课前至少知道下面这些内容。

### A. Python 基础

学生至少需要会：
- 变量赋值
- 数字、字符串、列表、字典
- `for` 循环
- `if` 判断
- 函数定义与调用
- `import`
- 基本报错阅读

最低掌握程度：
- 看得懂 `x = ...`
- 看得懂 `for ... in ...`
- 看得懂 `def ...`
- 能复制并运行简单代码块

### B. pandas / DataFrame 基础

这门课大量依赖 pandas，所以建议学生至少知道：
- `DataFrame` 和 `Series` 是什么
- 列选择：`df["col"]`
- 行列排序：`sort_values`
- 分组：`groupby`
- 滚动窗口：`rolling`
- 平移：`shift`
- 索引与列的区别
- `reset_index`
- 缺失值 `NaN`

最低掌握程度：
- 知道 DataFrame 就像一张表
- 知道列名和索引不是一回事
- 知道 `groupby` 是“先分组，再分别计算”

### C. Jupyter Notebook 基础

学生最好会：
- 打开 notebook
- 运行一个 cell：`Shift + Enter`
- 重新运行所有 cell
- 看 markdown 和 code cell 的区别
- 修改代码后重新运行

### D. 环境与包管理

建议学生知道：
- `pip install` 的作用
- 虚拟环境不是必须，但知道更好
- 包装不上时要先看报错，再检查 Python 版本和网络

推荐但不强制：
- 学会使用 `python -m pip install ...`

### E. Git 与代码仓库

对这套课来说，`git` 不是第一次课的硬性要求，但建议会最基本的：
- 什么是 repository / repo
- `git clone`
- 能把课件仓库拉到本地
- 知道“不要乱删文件、不要覆盖 notebook”

如果完全不会 git，也不影响第一次课，只要能拿到 notebook 文件即可。

### F. 量化研究最小概念

建议第一次课前先知道这些词，不要求精通：
- 股票
- ETF
- 指数
- 行情数据
- 财务数据
- 因子
- 回测
- 组合

### G. 后续课程会逐步接触的额外依赖

根据当前 `compact_6lectures`：
- 第 1-5 讲核心依赖：`pandas`、`numpy`、`matplotlib`、`rqdatac`
- 第 6 讲额外依赖：`scipy`

推荐一次性安装：

```bash
pip install pandas numpy matplotlib jupyter rqdatac scipy
```
