# FEC Quant Investing Workshop

这是一个面向本科生的量化投资入门 workshop 项目，围绕 **数据获取 -> 因子研究 -> 回测分析 -> 组合构建** 这条主线，组织了 6 节可运行的 Jupyter Notebook 课件。

项目目标不是堆砌模型或追求复杂度，而是把量化研究里最核心的流程拆成一套可以讲、可以跑、可以复现的实践材料。整套内容适合作为社团 workshop、课程实践或量化入门自学资料。

## 项目内容

核心课件位于 `quant_investing/compact_6lectures/`：

- `C01_数据入口与资产池.ipynb`
- `C02_财务衍生宏观因子.ipynb`
- `C03_回测闭环与RQAlpha模板.ipynb`
- `C04_技术指标与分散化.ipynb`
- `C05_单因子到多因子.ipynb`
- `C06_组合优化与收官.ipynb`

辅助文件：

- `讲座前置内容清单.md`：适合发给学生的课前准备说明

## 课程主线

整套 workshop 以一个清晰的研究闭环组织：

1. **先拿到可研究的数据**
2. **从数据中提取可解释的因子和特征**
3. **把信号接到最小可用回测框架**
4. **从单策略扩展到多因子和多资产组合**
5. **最终进入组合构建与权重优化**

对应到具体课件：

### C01 数据入口与资产池

重点内容：

- 使用 `rqdatac.get_price` 获取交易数据
- 理解 `close`、`prev_close`、`volume` 等基础字段
- 理解 `adjust_type='none' / 'pre'` 的口径差异
- 使用 `groupby`、`shift`、`rolling` 计算收益与滚动指标
- 过滤 ST、停牌与低流动性股票，构建 `tradable_pool`

这一节的目标是把“研究对象”和“数据口径”先定义清楚。

### C02 财务衍生宏观因子

重点内容：

- 使用 `get_pit_financials_ex` 读取原始财务报表字段
- 使用 `get_factor` 获取估值、成长与衍生因子
- 理解 `ttm / lyr / mrq` 的含义和使用边界
- 从价格序列构造动量、波动率等简单衍生特征
- 使用宏观数据补充市场状态信息
- 将多来源特征拼接为统一的 `factor_panel`

这一节的目标是把“原始数据”转化为“研究输入”。

### C03 回测闭环与 RQAlpha 模板

重点内容：

- 理解回测最小闭环：调仓日、权重、收益、净值
- 构建月频调仓的动量排序示例
- 从单资产收益累积到组合净值
- 加入手续费与换手成本的最小摩擦模型
- 理解研究环境与 `RQAlpha` 回测框架之间的映射关系

这一节的目标是让学生真正理解“净值曲线是怎么被算出来的”。

### C04 技术指标与分散化

重点内容：

- 以 Bollinger Band 为主案例讲解技术指标策略
- 以 RSI 作为对照策略展示不同指标语言
- 将单一信号扩展到多资产等权组合
- 比较单资产策略与分散化策略的净值差异

这一节的目标是从“单个信号”走向“多资产策略组合”。

### C05 单因子到多因子

重点内容：

- 在横截面上构造月频因子数据
- 使用 `winsorize` 和 `zscore` 做因子清洗与标准化
- 使用 RankIC 和分组收益评价单因子有效性
- 将多个因子整合为统一预测信号
- 把因子预测结果重新接回回测流程

这一节的目标是建立一套最小可用的多因子研究流程。

### C06 组合优化与收官

重点内容：

- 构建等权组合作为基准
- 使用 `scipy.optimize` 实现夏普最大化权重
- 通过滚动窗口估计收益与协方差
- 对比等权与优化组合的净值和波动
- 观察优化权重随时间的变化

这一节的目标是把研究结果进一步转化为投资组合权重。

## 适合的使用方式

这个项目既可以作为 workshop 课件，也可以作为自学路线图。

### 作为讲座课件

建议每节控制在 40-60 分钟：

- 前 10 分钟：讲概念和目标
- 中间 20-30 分钟：现场运行关键代码
- 最后 10 分钟：做小练习或总结

### 作为自学材料

建议按顺序学习：

1. 先跑通 `C01` 和 `C02`
2. 再看 `C03` 理解最小回测闭环
3. 之后学习 `C04-C06` 理解策略、因子和组合优化

不建议直接从 `C05` 或 `C06` 开始，因为它们默认你已经理解前面的数据与回测基础。

## 环境与依赖

建议使用：

- Python 3.10+
- Jupyter Notebook / JupyterLab
- `pandas`
- `numpy`
- `matplotlib`
- `scipy`
- `rqdatac`

部分课件还会涉及：

- `scikit-learn`
- `rqalpha`（主要是模板理解，不一定要求每节都现场运行）

示例安装：

```bash
pip install pandas numpy matplotlib scipy scikit-learn jupyter openpyxl
```

如果需要在线运行数据接口，还需要本地可用的 `rqdatac` 环境与对应权限。

## 使用说明

### 1. 打开课件

进入 `quant_investing/compact_6lectures/` 后，按顺序打开 `C01` 到 `C06`。

### 2. 初始化数据接口

课件默认使用在线数据接口，首个代码单元通常会包含：

- 时间区间配置
- 演示资产列表
- `rqdatac` 初始化

运行前请先确认本机的 `rqdatac` 许可可用。

### 3. 先理解再运行

每份 notebook 不是单纯的“代码集合”，而是按下面结构组织的：

- 课程目标
- 概念解释
- 关键代码
- 图表观察
- 小练习
- 小结

推荐先阅读 Markdown，再运行对应代码单元。

## 面向对象

这套材料主要面向：

- 对量化投资感兴趣的本科生
- 刚开始接触金融数据与回测的学习者
- 想从“模型/策略名词”回到“研究流程本身”的初学者

默认读者具备：

- 基本 Python 语法
- 基本的 `pandas` 使用经验
- 对收益率、波动率、因子、回测有初步概念

如果完全没有 Python 基础，建议先阅读 `讲座前置内容清单.md`。

## 这套课件强调什么

这套 workshop 有三个明确取向：

1. **流程优先**
- 先把数据、研究、回测、组合的关系讲清楚

2. **最小可运行**
- 每节尽量保留可执行的核心代码，而不是只讲概念

3. **可解释性优先**
- 重点解释为什么这样做、口径差异在哪里、哪些地方容易踩坑

因此，它不是一套追求模型复杂度的课程，而是一套追求研究逻辑完整性的入门材料。
